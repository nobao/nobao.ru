Начав изучение JavaScript, я понял, что без знания HTML и CSS далеко не продвинусь.

Сегодня хочу рассказать как я освежаю в памяти основные понятия и принципы HTML и CSS. Последние три дня я этим плотно занимаюсь. Вечером после работы я смотрю видео на канале YouTube Хауди Хо™ под названием:

  • Учим HTML за 1 час! #От Профессионала
  • Учим CSS за 1 час! #От Профессионала

А днем, в свободное время, в приложении для Android SoloLearn изучаю курсы по HTML и CSS. Получился довольно неплохой и продуктивный симбиоз: для получения базовых знаний вполне достаточно.
